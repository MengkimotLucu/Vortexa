# Dokumentasi Jasa & Harga Software QA Testing — LUMOVELO

Layanan pengujian kualitas perangkat lunak (*Software Quality Assurance & Testing*) Lumovelo bertujuan memastikan website, landing page, dan aplikasi mobile (Android & iOS) klien berjalan lancar, aman, bebas dari bug, dan siap untuk skenario beban traffic tinggi.

---

## 1. Skema Paket Layanan & Harga

### Paket 1: Basic (Smoke Audit)
*   **Investasi (IDR):** Rp950.000 (Sekali bayar)
*   **Investment (USD):** $69 (One-time)
*   **Deskripsi:** Audit cepat fungsionalitas alur utama halaman tunggal/landing page.
*   **Cakupan Fitur & Benefit:**
    *   Pengujian cepat alur utama (*Critical Path/Smoke Testing*).
    *   Audit responsivitas tampilan pada 2 browser utama (Chrome & Safari).
    *   Verifikasi link rusak (*broken links*) dan validasi input formulir kontak/konten.
    *   Laporan buglist dasar (*Excel/PDF Buglist*).
    *   Cocok untuk: Peluncuran landing page UKM, web portofolio sederhana.

### Paket 2: Pro (Multi-Device QA)
*   **Investasi (IDR):** Rp2.450.000 (Sekali bayar)
*   **Investment (USD):** $169 (One-time)
*   **Deskripsi:** Pengujian fungsionalitas menyeluruh di berbagai tipe perangkat fisik riil.
*   **Cakupan Fitur & Benefit:**
    *   Pengujian fungsionalitas *End-to-End* alur bisnis penuh (maks 8 halaman/modul).
    *   Pengujian kompatibilitas visual & fungsi di 5+ HP fisik Android & iOS asli.
    *   Pengecekan alur checkout keranjang dan integrasi payment gateway.
    *   Pencatatan bug detail disertai video rekaman + tangkapan layar langkah reproduksi.
    *   Gratis 1x verifikasi ulang (*re-testing*) setelah perbaikan oleh developer.
    *   Cocok untuk: Aplikasi mobile MVP, e-commerce menengah, portal web bisnis.

### Paket 3: Elite (Performance & API)
*   **Investasi (IDR):** Rp4.250.000 (Sekali bayar)
*   **Investment (USD):** $289 (One-time)
*   **Deskripsi:** Pengujian beban server, performa aplikasi, dan kestabilan integrasi API.
*   **Cakupan Fitur & Benefit:**
    *   Pengujian beban server (*Load Testing*) hingga 1.000 pengguna virtual bersamaan (k6 / JMeter).
    *   Uji responsivitas, latency, dan reliabilitas endpoint API.
    *   Analisis ketahanan memori (*memory leak*) & konsumsi baterai aplikasi.
    *   Pengecekan keamanan dasar (*basic security headers*).
    *   Laporan analisis rekomendasi optimasi kapasitas server & query database.
    *   Cocok untuk: Layanan SaaS, e-commerce skala menengah, web traffic tinggi.

### Paket 4: Custom (Automation & Security)
*   **Investasi (IDR):** Mulai Rp9.500.000
*   **Investment (USD):** Mulai $649
*   **Deskripsi:** Otomatisasi pengujian regresi penuh dan audit keamanan tingkat lanjut.
*   **Cakupan Fitur & Benefit:**
    *   Pembuatan skrip testing otomatis menggunakan Cypress, Playwright, atau Appium.
    *   Integrasi otomatisasi tes ke dalam alur pipeline CI/CD (GitHub Actions / GitLab CI).
    *   Uji penetrasi keamanan aplikasi (*Basic Pentest* berbasis OWASP Mobile/Web Top 10).
    *   Handoff source code skrip tes dan sesi pelatihan untuk tim developer internal.
    *   Gratis bantuan pemeliharaan skrip otomatisasi selama 1 bulan pertama.
    *   Cocok untuk: Aplikasi finansial (fintech), super-app, sistem korporasi besar.
